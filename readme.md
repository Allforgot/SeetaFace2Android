# SeetaFace2Android

SeetaFace2 origin project: [github](https://github.com/seetafaceengine/SeetaFace2)

## SeetaFace2AndroidLib

Generate Android Libs

### Getting Started

#### Installation

```bash
export ANDROID_NDK_HOME=path_to_android_ndk_dir
cd SeetaFace2AndroidLib && ./build.sh
```
